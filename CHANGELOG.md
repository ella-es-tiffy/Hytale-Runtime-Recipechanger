# Changelog - RtChanger

## [1.0.0] - 2026-01-20
### Added
- Initial release of **RtChanger**.
- Support for overriding any Item ID via `rt_overrides.json`.
- 1:1 AssetStore synchronization logic (UI & Logic sync).
- Successor to the Proof of Concept (POCMod) system.
- Auto-migration for legacy `poc_config.json` and `recipe_overrides.json` files.
- Automatic pattern matching for Hytale "Generated" recipes.
- Scheduled retry mechanism (5s and 15s) to catch late-loading mod assets.
